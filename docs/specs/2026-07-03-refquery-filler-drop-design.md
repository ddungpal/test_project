# refYouTubeQuery 의문형 필러 제거 (refquery-filler-drop)

작성일: 2026-07-03

## 배경 / 문제

직전 `onboarding-regenerate-staleness` step0에서 refYouTubeQuery를 4토큰 캡으로 고쳐 "0개 반환"은 해결했으나,
남은 4토큰 `"커버드콜 ETF가 대체 뭐길래"`가 **여전히 저성과 영상**을 반환한다(라이브 재생성 결과: 734·1,673·164·5,731 조회).

## 🔬 근본 원인 (라이브 API 측정으로 확정)

같은 주제로 쿼리 변형을 실제 검색해 상위 조회수를 측정:

| 쿼리 | 상위 조회수 |
|---|---|
| `커버드콜 ETF` | 482만·148만·89만·85만·68만 ✅ |
| `커버드콜 ETF가` (조사 O) | 148만·89만·68만·31만 ✅ |
| `커버드콜 ETF가 배당을` | 482만·209만·148만·123만 ✅ |
| `커버드콜 ETF가 **대체**` | 9만·1.5만·6천 ❌ |
| `커버드콜 ETF가 대체 **뭐길래**` | **734** ❌❌ |

**결론:**
- **조사("가")는 결과를 안 해친다** → 조사 제거 불필요(위험·단어 훼손: '국가'→'국').
- **의문형/강조 필러("대체","뭐길래")가 결과를 파괴한다** → 이것만 걷어내면 됨.

## 결정

`refYouTubeQuery`에 **필러 stopword 제거**를 추가하고 캡을 4→3으로. 조사 제거는 안 함(불필요·위험). 1 step·마이그0.

## 설계 — `refYouTubeQuery` 개정 (prepare.ts)

```ts
// 검색 가치 없는 의문/강조 필러 — 앞의 핵심 명사 뒤에 붙어 검색을 뒤틀어 저성과/무관 영상을 부른다.
//   (라이브 측정: '대체'·'뭐길래' 추가 시 482만→9만→734로 붕괴). 조사는 제외(결과 무해·단어 훼손 위험).
const REF_QUERY_FILLER = new Set([
  "대체","도대체","뭐길래","뭐길레","왜","진짜","정말","과연","그냥",
  "얼마나","어떻게","무엇","뭐","뭔데","이게","이거","이런","이렇게","도데체",
]);

export function refYouTubeQuery(topicTitle: string): string {
  const raw = (topicTitle ?? "").trim();
  let s = raw.replace(/\[[^\]]*\]/g, " ").replace(/\([^)]*\)/g, " ");
  s = s.split(/[,|\n?!]/)[0] ?? s;
  s = s.replace(/["'“”‘’]/g, " ").replace(/[.…~]+$/g, "").replace(/\s+/g, " ").trim();
  const MAX_TOKENS = 3; // 4→3(필러 제거 후 핵심 명사 위주로 더 짧게)
  const tokens = s.split(" ").filter((t) => t && !REF_QUERY_FILLER.has(t)); // ★ 필러 제거
  s = tokens.slice(0, MAX_TOKENS).join(" ");
  return s.length >= 2 ? s : raw; // 폴백 유지
}
```

- 예: `"커버드콜 ETF가 대체 뭐길래 배당을 10%씩 줄까? 초보를 위한 원리 완전 정복"`
  → 절 컷("?") → `"커버드콜 ETF가 대체 뭐길래 배당을 10%씩 줄까"` → 필러 제거 → `"커버드콜 ETF가 배당을 10%씩 줄까"`
  → 3토큰 → **`"커버드콜 ETF가 배당을"`** → 482만 조회 영상 반환.
- 콤마 제목 `"월배당 ETF, 매달…"` → `"월배당 ETF"`(불변).
- ★ 필러가 정확한 토큰 매치일 때만 제거(부분 문자열 아님) — '배당을'의 '을' 등 오제거 없음.
